# web-archive-collection
Коллекция архивов WARC сайтов гимназии и музеев
# Коллекция веб-архивов

**Автор:** Полина Попова  
**Описание:** Коллекция архивов WARC с сайтов гимназии, музея Гагарина и Нерчинского музея.  
Архивы сохранены с помощью утилиты [Metawarc](https://github.com/internetarchive/metawarc).  

## Список сайтов и архивов

| Архив | Сайт | Описание |
|-------|------|----------|
| gimn2-archive.warc.gz | Гимназия №2 | Полный сайт гимназии с разделами о школе, новостях, расписании и т.д. |
| gagarin-archive.warc.gz | Музей космоса в Гагарине | Главная страница музея с информацией о выставках и экспозициях. |
| nerchinsk-archive.warc.gz | Нерчинский музей | Страница с описанием музея и экспозиций. |

## Метод и технология сбора

- Архивирование проведено с использованием утилиты `metawarc`.  
- Формат хранения: WARC (Web ARChive).  
- Для анализа использовались команды `analyze`, `metadata`, `export`.
